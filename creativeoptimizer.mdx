Creative Optimizer<sup>BETA</sup> is a tool focused on reducing creative asset weight available as part of an ESP subscription or a standalone commercial license. 

To learn more or gain access, please speak to your account manager.


## Submitting video creatives for optimization

To optimize your creative, follow the steps below from the ***Creative Optimizer<sup>BETA</sup>*** section of the interface:

1. **Select your seat** - the *seat* is the contractual entity so most users will only have one available
2. **Select the advertiser**
3. **Select the platform** - we tailor optimization based on the platform so it is important to specify where the creative will be used
4. **Enter additional recipients** - if you need someone else to be able to download the optimized creatives, input their email so that they get the link directly
5. **Upload your creatives** - drag and drop or click to upload files. URLs cannot be used.

<Note> File uploads must be in .zip format containing only .mp4 or .mov files <br /><br />
The max file weight should be 500MB per upload and videos should be no longer than 3 minutes </Note>

6. **Push *Apply* to submit** - a confirmation screen will appear totalling the number of recognized creatives expected to be optimized along with any error messages.

7. **Confirm you wish to proceed** - the relevant credits will be consumed on confirmation and you will be provided with a request ID. Use this request ID if you need to refer to it later with support.

8. **Wait for e-mail** - you will receive a download link for your optimized creatives in your e-mail upon completion


## Optimized Creatives


Upon downloading the zip file containing your optimized creatives you will see several suffixes added to the filename:

e.g. 

[ORIGINALFILENAME] ***ctv-16.9-1080p-1785kbps-stereo-128kbps-most-optimized***

which corresponds to: 

[ORIGINALFILENAME] ***device-aspectratio-videoresolution-videobitrate-audiochannel-audiobitrate-optimization level***

These elements are described below


| Field | Description |
| --- | --- |
| Device | Intended viewing device: CTV, Desktop/Tablet, Smartphone, AllDevices |
| Aspect Ratio | Portrait / Landscape / Square ratios - 16:9, 9:16, 4:3, 3:4, 1:1, 4:5, 5:4|
| Video Resolution | Video dimensions: 1080p - 360p  |
| Video Bitrate | Bitrate of video in kbps: 1785kbps - 320kbps |
| Audio Channel | Mono or Stereo audio channels |
| Audio Bitrate | Bitrate of audio in kbps: 128kbps - 80kpbs|
| Optimization Level | Most - Least optimized and 'Recommended' |


These suffixes can help you understand the specific details of the optimized video.

**Best Practice would involve separating targeting by device** because the different screen sizes mean different levels of optimization are possible without affecting the user viewing experience. Therefore, **we recommend that users review the most optimized versions for each device type** and implement them separately where possible.

However, **we recognise it is not always possible to split out targeting and budget by device so we have also included a 'RECOMMENDED' creative that should perform well across all device types** without reduction in visual fidelity. Use this version if you are unable to split line items or campaigns by device.

<Note> Things to look for when reviewing video quality against the original: <br /><br /> -Subtitles / Smallprint
<br /> -Fast moving images
<br /> -Individual sound channels being used </Note>
